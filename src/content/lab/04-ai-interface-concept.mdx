A design exploration for a new kind of task interface: one where AI autonomy is an explicit, first-class control — not buried in settings, but inline with the task itself.

The concept: every task has an autonomy slider. "Suggest" mode shows AI recommendations that require explicit user approval. "Draft" mode has AI produce a complete first attempt for review. "Auto" mode delegates fully with a notification on completion.

Most current AI tools force a choice between "chatbot" and "autopilot." This explores the middle ground.

**Status:** Concept stage. Figma prototype in progress.
