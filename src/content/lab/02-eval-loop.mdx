A structured evaluation framework for reviewing AI-generated output. Instead of asking "does this look right," it asks specific questions: Does it match the spec? Does it handle the defined edge cases? Does it follow the established patterns? Does it introduce any regressions?

The framework makes review faster and more consistent — and makes feedback to the agent more specific, which improves subsequent iterations.

**Status:** Active. Used in every meaningful AI-assisted build.
